## Overview  
Employee Statistics Analysis is a data-driven project designed to uncover insights from workforce data using Python. It applies statistical methods like **T-Test** and **ANOVA** to analyze relationships between variables such as salary, education, experience, performance, and promotions. The goal is to support HR decisions through meaningful data interpretation.

## Objectives  
- Analyze relationships between education, salary, and performance.  
- Evaluate departmental salary differences using ANOVA.  
- Understand the impact of experience and promotion on retention.  
- Provide actionable insights for HR and management.
